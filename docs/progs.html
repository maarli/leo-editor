RED, CPP and CSTAR

I have written several programs that have been placed in the public domain. Among them are RED, CPP and CSTAR. These programs are available on CD ROM from The C/C++ Users Group:

Volume 318 contains RED, a text editor for MSDOS. RED was originally written in 1983.

Volume 319 contains CPP, an ANSI-Standard C preprocessor. CPP was written in 1987.

Volume 357 contains a compiler for a language I invented called CSTAR. For those of you with a very long memory, CSTAR is similar to pl/360 but with a C-style syntax. CPP was written in 1987. It has, at best, historical interest.
